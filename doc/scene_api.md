## `scene` module API

The scene module contains functionality for manipulating the scene as well as various objects for building a scene and displaying text on the screen.

The following classes are exposed to the Lua API:

* [Particle](#particle)
* [Vertex](#vertex)
* [Sphere](#sphere)
* [Mesh](#mesh)
* [TextProperties](#textproperties)
